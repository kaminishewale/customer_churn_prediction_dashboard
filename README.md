# Customer Churn Prediction and Analytics Dashboard

## Project Overview
This project predicts customer churn using Machine Learning and provides business insights through an interactive Streamlit dashboard.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Plotly

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest Classifier

Accuracy: 92.69%

## Project Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Churn Prediction
- Confusion Matrix Analysis
- Feature Importance Analysis
- Interactive Dashboard
